Hydrogen Fuel Cell Car

  • Gallery

Summary

The fuel cell car was designed to be an energy efficient autonomous vehicle that is able to navigate through an arbitrary white course marked with black guide lines. Optimization of the car included efficient algorithms to make well-informed decisions on forks and dead-ends, dynamic car designs, and proper cell maintenance.

Tech

Components:

TI MSP430 microcontroller • Pololu 6V motor driver • Two hydrogen fuel cells
• QTR-1A Reflectance Sensors • DC motors • Touch sensors

Software:

C++ • IAR embedded workbench

Next Project

Guitar Villain

  • Gallery

Summary

This project provides an alternative method to learning the basics of guitar playing. It is built in a similar fashion to the popular video game controller 'Guitar Hero', however in this version, the guitar can be set to an autonomous mode. Users can learn to play the guitar themselves, or learn from the guitar through listening to the pitches or play alongside the guitar as a duet.

Tech

Components:

NXT Mindstorm brick • 2 Motors (w/ encoders) • Ultrasonic sensor • Touch sensor • DC motors • Touch sensors

Software:

C++ • Mindstorm EV3 programming software
